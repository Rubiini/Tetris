# Tetris
Javalabra projekti

####Aiheen kuvaus
Tetris. Peli, jossa on tarkoituksena muodostaa putoavista palikoista vaakarivejä.
Pelaaja voi liikuttaa palikoita kääntämällä, sekä liikuttamalla vasemmalle tai oikealle nuolinäppäimillä.
